## 🔍 Поиск устройств
Приложение автоматически фильтрует найденные устройства: отображаются самокаты Ninebot/Xiaomi. 
В данный момент приложение стабильно подключается к самокатам **F3/F3 Pro**. Теоретически должно подключаться к моделям **G3** (не проверено). Поддержка других самокатов будет добавлена позднее.
- **Кнопка «Карта» (Глобус):** Позволяет пропустить поиск самоката и перейти на главный экран в офлайн-режиме.

## 🗺️ Главный экран (Карта и инструменты)
В данный момент для прокладки маршрута используются Яндекс Карты. Планируется добавление других сервисов в будущем.

### Панель маршрутов 
*(Появляется под поиском после тапа по карте или выбора места из списка)*
- 🚗 **Авто:** Строит автомобильный маршрут до точки.
- 🚶 **Пешком:** Из-за проблем с API Яндекса для пешеходов, в данный момент строится велосипедный маршрут.
- 🚲 **Велосипед:** Строит велосипедный маршрут по тропинкам и наземным переходам (почти не отличается от пешеходного).

### Правая панель (Инструменты карты)
- 🚦 **Пробки:** Включает/выключает отображение пробок на дорогах.
- ☁️ **Погода:** Загружает точный прогноз погоды *(источник данных можно изменить в настройках)*. Внимание: некоторые источники в МСК работают только через Wi-Fi из-за блокировок мобильных операторов.
- 🧭 **Компас:** Появляется при вращении карты, возвращает положение «Север сверху».
- 🎯 **Моя геолокация:** Перемещает камеру к текущему местоположению (синий кружочек «Я»).
- ⏱️ **Дашборд:** Открывает полноэкранную панель приборов самоката.
- ⏺️ **Запись трека:** Запускает фоновую запись маршрута. Пишет данные GPS и телеметрию самоката (скорость, заряд, температуру компонентов).

## 🏍️ Дашборд (Панель приборов)
- **Управление:** Кнопка включения и выключения самоката.
- **6 блоков данных:** Кликом по любой карточке можно выбрать, какую информацию она будет отображать (ток, напряжение, пробег, температуры и т.д.).

## ⚙️ Меню Настроек
- **Записи треков:** Открывает список всех сохраненных маршрутов поездок.
- **Настройки приложения:** Выбор источника данных для показа погоды.
- **Настройки ТС:** Управление функциями самоката *(в данный момент активна только часть функций, остальные тестируются и станут доступны позже)*.
- **Обратная связь:** Переход к форме отправки баг-репортов/предложений.
- **О приложении:** Версия программы и ссылка на Политику конфиденциальности.

## 🛤️ Список треков и Плеер маршрутов

### В списке треков:
Поддерживается выделение нескольких файлов галочками для массовых действий.
- **GPX:** Экспорт маршрута в универсальный GPS-формат (подходит для Strava, Яндекс.Карт и т.д.). Только геоданные.
- **Поделиться:** Отправка оригинального файла `.pdash` (геоданные + телеметрия) через мессенджеры.
- **Удалить:** Массовое удаление записей.

### Плеер записи и редактор маршрутов:
В плеере записи (справа сверху) находится **Гаечный ключ** — открывает меню, где можно включить **«Пропуск стоянок»** и отключить всплывающие подсказки редактора.

Полностью переработанная логика редактирования `.pdash` файлов:
- ✏️ **Редактирование (Синий карандаш):** Открывает режим ручной корректировки. Здесь вы можете вручную удалить неправильно отрисованные точки GPS и добавить новые, прокладывая правильный маршрут.
- ➡️ **Далее (Зеленая стрелка):** Переход к автоматической корректировке.
- 🔍 **Поиск (Лупа):** Приложение автоматически предложит отредактировать куски вашего трека с помощью технологий Яндекса. *(Кнопка "Назад" вернет к ручному редактированию точек).*
- 💾 **Сохранить (Зеленая дискета):** Синхронизирует все оригинальные данные трека и телеметрии с новым маршрутом и сохраняет трек в новый файл.
