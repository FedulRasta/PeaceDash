# PeaceDash

PeaceDash — это альтернативное приложение для управления, мониторинга, навигации и записи треков поездок для электросамокатов.

## Поиск устройств
Приложение автоматически фильтрует найденные устройства: отображаются самокаты Ninebot/Xiaomi. 
В данный момент стабильно поддерживается подключение к самокатам **F3/F3 Pro**. Теоретически поддерживается серия **G3** (не проверено). Поддержка других самокатов будет расширяться.
- **Кнопка «Карта» (Глобус):** Позволяет пропустить поиск самоката и перейти на главный экран в офлайн-режиме.

## Главный экран (Карта и инструменты)
Для прокладки маршрутов и навигации используется API Яндекс Карт.

### Панель маршрутов 
*(Появляется под поиском после тапа по карте или выбора места из списка)*
- **Авто:** Строит автомобильный маршрут до точки.
- **Пешком:** Строит пешеходный маршрут.
- **Велосипед:** Строит велосипедный маршрут по тропинкам и наземным переходам.

### Правая панель (Инструменты карты)
- **Пробки:** Включает/выключает отображение пробок на дорогах.
- **Погода:** Загружает точный прогноз погоды *(источник данных можно изменить в настройках)*. Внимание: некоторые источники в МСК работают только через Wi-Fi из-за блокировок мобильных операторов.
- **Компас:** Появляется при вращении карты, возвращает положение «Север сверху».
- **Моя геолокация:** Перемещает камеру к текущему местоположению (синий кружочек «Я»).
- **Навигация (Значок маршрута):** Передача навигационных подсказок текущего маршрута на дисплей самоката (направление, улица, дистанция до маневра). *Примечание: Для корректной работы требуется включить функцию Bluetooth автомобиля в официальном приложении Ninebot и выполнить сопряжение с самокатом.*
- **Дашборд:** Открывает полноэкранную панель приборов самоката.
- **Запись трека:** Запускает фоновую запись маршрута (с улучшенным алгоритмом сглаживания GPS). Пишет геоданные и телеметрию самоката (скорость, заряд, температуры). Все файлы треков сохраняются в общую директорию `Download/PeaceDash`, что предотвращает их потерю при удалении приложения.
## Дашборд (Панель приборов)
- **Управление:** Кнопка включения и выключения самоката.
- **6 блоков данных:** Кликом по любой карточке можно выбрать отображаемую информацию (ток, напряжение, пробег, температуры и т.д.).

## Меню Настроек
- **Записи треков:** Открывает список всех сохраненных маршрутов поездок.
- **Настройки приложения:** Выбор источника данных для показа погоды и ручная установка средней скорости для пешеходных/велосипедных маршрутов (необходимо для точного расчета времени при навигации).
- **Настройки ТС:** Управление функциями самоката.
- **Обратная связь:** Переход к форме отправки баг-репортов/предложений.
- **О приложении:** Версия программы и ссылка на Политику конфиденциальности.

## Список треков и Плеер маршрутов

### В списке треков:
Поддерживается выделение нескольких файлов галочками для массовых действий.
- **GPX:** Экспорт маршрута в универсальный GPS-формат (подходит для Strava, Яндекс.Карт и т.д.). Только геоданные.
- **Поделиться:** Отправка оригинального файла `.pdash` (геоданные + телеметрия) через мессенджеры.
- **Удалить:** Массовое удаление записей.

### Плеер записи и редактор маршрутов:
В плеере записи (справа сверху) находится **Гаечный ключ** — открывает меню, где можно включить **«Пропуск стоянок»** и отключить всплывающие подсказки редактора.

Логика редактирования `.pdash` файлов:
- **Редактирование (Синий карандаш):** Открывает режим ручной корректировки. Позволяет вручную удалять неточно отрисованные точки GPS и добавлять новые.
- **Далее (Зеленая стрелка):** Переход к автоматической корректировке.
- **Поиск (Лупа):** Автоматическое выравнивание отрезков трека по дорогам с помощью технологий Яндекса. *(Кнопка "Назад" вернет к ручному редактированию точек).*
- **Сохранить (Зеленая дискета):** Синхронизирует все оригинальные данные телеметрии с новым (выровненным) маршрутом и сохраняет трек в новый файл.
